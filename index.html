🔗 Cara Menghubungkan ke Google Sheets

  1. Buka script.google.com dan buat proyek baru.
  2. Paste kode Apps Script dari file google-apps-script.js yang disertakan.
  3. Deploy sebagai Web App (Execute as: Me, Who has access: Anyone).
  4. Salin URL deployment dan paste ke variabel GOOGLE_SHEET_URL di app.js.
  5. Data akan otomatis tersimpan ke Google Sheets setiap kali formulir disubmit.
